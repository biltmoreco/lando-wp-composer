# Lando WP Composer

Build or start up Lando

`lando start`

---

Install a new plugin

`lando composer require "wpackagist-plugin/where-used":"^1"`

---

Get versions for running services

`lando versions`

---

Composer install

`lando install`

---

Composer install (dry run)

`lando install:dry`
